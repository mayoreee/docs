# JMES.js

JMES.js can be used to interact with cosmos blockchains like JMES within JavaScript runtimes, such as Node.js and the browser. feather.js enables the following functionalities:

- Deserializing blockchain data into JavaScript objects with native data types and methods.
- Serializing objects back into a blockchain-compatible format.
- Providing access to the JMESd node API (LCD) and other compatible feather chains from a JavaScript-based interface.
- Providing additional utilities, such as hash functions and key-signing algorithms.
