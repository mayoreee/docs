### Download LocalJMES and start building <ThemedImage sources={{light:"/img/icons/localterra.svg", dark:"/img/icons/dark/localterra.svg"}} height="30px" />

LocalJMES is a scaled-down JMES blockchain that allows you to test your dApp locally. LocalJMES provides a single validator node and [10 preconfigured wallet accounts](./localterra/accounts.mdx).

## Development Suite

Congrats! You have the basics down!

Now you can dive into the other tools in the JMES Development Suite. Visit the [tool guide](which-tools.mdx) for more info.

### JMES.js 

Use feather.js to create bots, power NFT mints, and for all-purpose back end services. Visit the [JMES.js repository](https://github.com/jmesworld/jmes.js) to get started.

### JMES Discord Server

If you would like to connect with the developer community and ask questions related to software development on JMES, join the [JMES Discord server](http://discord.jmes.cloud). 

### Dev Incubator

Join the [JMES Developer Incubator](http://incubator.jmes.cloud), a blockchain-funded DAO offering JMES for code contributions using micro-incentives.
